# spacex

## Run Development server

Run `npm start` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.

## Angular Universal Build

Run `npm run build:ssr` to build the project using angular universal. The build artifacts will be stored in the `dist/server` directory..

## Run with Angular Universal

Run `npm run serve:ssr` for a dev server. Navigate to `http://localhost:4000/`. The app will automatically reload if you change any of the source files.

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).

## Running end-to-end tests

Run `ng e2e` to execute the end-to-end tests via [Protractor](http://www.protractortest.org/).
